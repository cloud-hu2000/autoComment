Auto Register Filler 设置

在这里安全地保存你的 DashScope / 通义千问 API Key，并自定义网站推广 Skill 语言模板。
- 只保存在浏览器的 chrome.storage.sync 中，本扩展不会上传到任何第三方服务器。
- 修改后，重新打开页面或重新触发右键菜单即可生效。
- 支持多行文本，将作为调用通义千问时的 system 提示词。
- 为空时会使用扩展内置的默认模板；修改后立即生效。
- 这里配置的内容会用于注册表单和评论表单中的 Website / URL / Name / Email / Password / 网址 / 姓名 / 邮箱 / 密码 等字段。
- 在评论表单中自动填充到 Website / URL / 网址 字段。
- 为空时不会自动填 Website 字段。
- 用于自动填写注册表单和评论表单中的 Name / Email / Password / 姓名 / 邮箱 / 密码 字段。
- 姓名和邮箱留空时会使用扩展内置的默认值；密码留空时会使用扩展内置的默认密码。
- 勾选后，每次打开页面都会在右下角自动弹出推广助手窗口，方便连续撰写评论。
- 取消勾选则仅在点击扩展图标时才会打开浮动窗口。
- 该开关默认关闭，并且每次重新打开浏览器时都会自动恢复为关闭状态。
- 仅在勾选后，扩展才会在检测到评论框时自动调用通义千问生成推广文案，以避免在不知情的情况下持续消耗 token。
